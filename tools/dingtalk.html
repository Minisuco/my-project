1
创建企业组织
钉钉首页 → 工作台 → 通讯录 → 创建企业，填写"功夫之星武道馆"，上传营业执照完成认证
2
建立部门层级
管理后台 → 组织管理 → 部门管理 → 新建三个子部门：省锡中店 / 惠山体育馆店 / 锡山天一店，各自设置负责人
3
添加员工并分配部门
通过手机号邀请员工加入，填写姓名、岗位、所属部门。建议同时填写工号，方便后续考勤和薪酬对应
4
设置管理员权限
校长为超级管理员，各校区负责人设为子管理员（只能管理本校区数据），教练/前台为普通成员
2
考勤打卡设置
按校区分组设置上下班时间，支持GPS定位打卡
⏰
📍
GPS定位打卡 推荐
后台 → 考勤打卡 → 考勤组管理 → 新建考勤组 → 打卡方式选"地点打卡" → 输入各校区地址，设置打卡范围200米以内有效
优点：防止代打卡，教练在校区外无法打卡
🕐
上下班时间设置
建议按以下时间段配置（可根据实际调整）：
· 工作日班：上午 09:00 上班，21:00 下班
· 周末班：09:00 上班，18:00 下班
· 允许弹性打卡30分钟，迟到超30分钟记异常
👥
按校区分组设置
每个校区建立独立考勤组，负责人可单独查看本校区打卡数据，校长可查看全部
各校区地址：省锡中店 / 惠山体育馆 / 天一体育馆
📊
考勤报表导出
每月1号自动生成上月考勤汇总，导出Excel用于核算工资，路径：考勤 → 统计报表 → 月度报表
岗位建议考勤组打卡要求
教练各自所在校区组上课前15分钟打卡，下课后打卡
前台各自所在校区组正常上下班时间打卡
校长总部管理组（不限位置）外勤打卡即可
3
审批流程配置
请假、报销、采购等一键线上审批，留有记录
✅
审批类型适用场景审批流程建议
请假申请 教练/前台请假 本人申请 → 校区负责人审批 → 校长知会 设置最少提前1天申请
外出申请 比赛带队、外出拜访 本人申请 → 校长审批 填写外出事由和预计返回时间
费用报销 采购耗材、比赛费用 本人申请（附发票照片）→ 校长审批 50元以下负责人可直接批
物品采购 道服、护具、耗材 校区负责人申请 → 校长审批 备注用途和数量
调班申请 教练换课 本人申请 → 负责人审批 注明换课原因和换班人员
配置路径：管理后台 → 工作台 → 审批 → 创建审批模板 → 选择对应类型并配置审批人层级
小技巧：开启"审批超时自动提醒"，校长24小时未处理自动提醒，避免卡单
4
日志 / 周报汇报
教练每日上报带课情况，负责人周报总结校区数据
📝
📋
教练日志模板（每日下课后填写）
· 今日带课班级：__班 共__人
· 课程内容简述：（简单写本节课教学重点）
· 学员出勤异常：（缺课学员姓名）
· 家长沟通记录：（沟通了哪位家长，内容）
· 明日带课计划：（明天几点什么班）
📊
校区负责人周报模板（每周一填写上周数据）
· 本周课时完成情况：计划__节 / 实际完成__节
· 本周新增学员：__人
· 本周续费学员：__人 / 金额__元
· 本周退费情况：__人 / 原因
· 下周重点工作计划：（列1-3条）
设置路径：工作台 → 日志 → 新建日志模板 → 按上方字段配置问题项，设置每日/每周提醒
汇报对象：教练日志汇报给校区负责人；负责人周报汇报给校长
建议规则：每日21:30前提交日志，超时未交由负责人催促
5
任务与待办管理
重要事项派发任务，有截止时间，有责任人
✔️
🎯
任务派发规范
发任务时必填三要素：任务名称 + 截止时间 + 负责人
路径：钉钉首页 → 待办 → 新建任务 → 指定执行人
执行人完成后标记"已完成"，校长可看到完成状态
📌
适合用任务管理的场景 武道馆专用
· 考级通知发布（截止：考级前5天）
· 赛事报名收集（截止：报名截止日前1天）
· 道服/护具采购（截止：下次上课前）
· 月度绩效数据汇总（截止：每月5号）
· 新学员入档建档（截止：入学当日）
⏰
超时提醒设置
创建任务时开启"到期提醒"，截止前1天自动提醒负责人，确保不遗漏
6
群组规范管理
建议建立标准群组体系，信息分流，不混乱
💬
群名称成员用途群规则
功夫之星 · 管理层 校长 + 三校区负责人 重要决策、数据汇报、紧急通知 仅限管理层，24小时必回
省锡中店 · 全体 该校区全员 校区内部通知、排班、日常沟通 工作时间发言，禁止闲聊
惠山体育馆店 · 全体 该校区全员 同上 同上
天一体育馆店 · 全体 该校区全员 同上 同上
功夫之星 · 全员大群 所有员工 全公司通知、表彰、重大公告 仅管理员发言，其他人禁言
教练交流群 所有教练 教学经验分享、课程研讨 正能量，禁止抱怨
📢
群公告使用规范
重要通知必须用群公告（非普通消息），成员必须"已读确认"
适合发群公告的内容：考级通知、比赛信息、放假安排、重要制度变更
🔕
全员大群禁言设置
路径：群设置 → 群管理 → 全员禁言 → 开启
仅校长和管理员可发言，防止信息混乱，确保每条通知都被看到
📎
群文件夹管理
各群内建立群文件夹，统一存放：课表、考级资料、活动照片、通知模板
路径：群 → 右上角 → 文件 → 新建文件夹分类存放
